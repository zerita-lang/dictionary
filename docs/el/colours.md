# Χρώματα

Στην ζερίτα υπάρχουν 13 βασικά χρώματα:

| Ζερίτα | Ελληνικά  |
| ------ | --------- |
| blank  | άσπρο     |
| ner    | μαύρο     |
| ruj    | κόκκινο   |
| verd   | πράσινο   |
| azul   | μπλε      |
| flav   | κίτρινο   |
| brun   | καφέ      |
| purpur | μωβ       |
| roz    | ροζ       |
| bej    | μπεζ      |
| cian   | γαλάζιο   |
| oranj  | πορτοκαλί |
| griz   | γκρι      |

There are two prefixes for the colour shade:

| Ζερίτα | Ελληνικά |
| ------ | -------- |
| lum    | ανοιχτό  |
| skur   | σκούρο   |

and four adjectives for the colour finish:

| Ζερίτα  | Ελληνικά   |
| ------- | ---------- |
| metalik | μεταλλικό  |
| mat     | ματ        |
| glos    | γυαλιστερό |
| neon    | νέον       |

## Examples

1.  _azul glos_

    -   **μετάφραση:** γυαλιστερό μπλε

1.  _skurruj_

    -   **μετάφραση:** σκούρο κόκκινο

1.  _lumverd_

    -   **μετάφραση:** ανοιχτό πράσινο
