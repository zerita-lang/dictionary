# Colours

There are 13 base colours in Zerita:

| Zerita | English |
| ------ | ------- |
| blank  | white   |
| ner    | black   |
| ruj    | red     |
| verd   | green   |
| azul   | blue    |
| flav   | yellow  |
| brun   | brown   |
| purpur | purple  |
| roz    | pink    |
| bej    | beige   |
| cian   | cyan    |
| oranj  | orange  |
| griz   | grey    |

There are two prefixes for the colour shade:

| Zerita | English |
| ------ | ------- |
| lum    | light   |
| skur   | dark    |

and four adjectives for the colour finish:

| Zerita  | English  |
| ------- | -------- |
| metalik | metallic |
| mat     | matte    |
| glos    | glossy   |
| neon    | neon     |

## Examples

1.  _azul glos_

    -   **translation:** glossy blue

1.  _skurruj_

    -   **translation:** dark red

1.  _lumverd_

    -   **translation:** light green
